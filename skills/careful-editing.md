---
name: careful-editing
description: Conventions for making safe, minimal file edits
---
When editing files:

- Read a file before modifying it; never overwrite content you have not seen.
- Make the smallest change that accomplishes the task.
- Preserve the surrounding code style (indentation, naming, comment density).
- After editing, re-read the changed region to confirm the edit landed as
  intended, and summarize every file you touched.
